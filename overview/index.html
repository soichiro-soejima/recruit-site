1ページで分かる、アノマリーマーケティングの魅力

事業概要

Web広告運用を軸とした、
マーケティング支援会社

アノマリーマーケティングは、リスティング広告・SNS広告の運用代行を中心に、LP制作・CRM支援・コンテンツ制作まで、Webマーケティング全般を手がける会社です。

一般的な広告代理店は、リード（問い合わせ・資料請求）の獲得までをゴールとします。しかし実際にクライアントが必要としているのは、リードではなく契約であり、売上です。

私たちはその最終ゴールまで責任を持ちます。広告の設計から、LP改善、営業支援まで一気通貫で関わることで、クライアントの事業成果に直結した支援を行っています。

02 —— 3 FEATURES

3つの特徴

01

ALL IN ONE

一気通貫で、事業全体に関わる

営業・提案・広告運用・LP改善・CRM支援まで、同一のコンサルタントが担当します。分業体制ではないため、クライアントの事業全体を視野に入れながら仕事ができるのが特徴です。広告の数字だけでなく、売上・契約数という最終ゴールに向き合う経験を積みながら、一般的な広告運用者では身につかない事業視点が自然と鍛えられます。

02

DIRECT REWARD

実力主義の環境

担当案件の粗利30%がそのまま月収になります。評価面談も、上司への根回しも、年次も関係ありません。成果を出した人が正当に報われる仕組みをシンプルに実現しています。評価制度に時間を使う必要がないぶん、クライアントへの貢献に集中できます。

03

SMALL & FAST

少数精鋭で、裁量が大きい

少人数だからこそ、一人が担う領域が自然と広がります。広告運用にとどまらず、営業・提案・改善まで一気通貫で関わるのが特徴です。一人ひとりの影響力も大きく、自分のアイデアや行動が組織に直接影響するフェーズで働けます。大企業のように「稟議を通す」「上の判断を待つ」といったことが少なく、マーケター以上の総合力が身につく環境です。

03 —— CULTURE CODE

カルチャーコード

なぜ、カルチャーコードをつくったのか。

入社後に「思っていたのと違う」とならないために、私たちは働く上での価値観をすべて言語化しました。何を大切にするのか、何を許容しないのか、どんな姿勢で仕事に向き合うのか。選考前に読んでもらうことで、お互いにとって後悔のない判断ができると考えています。
